What is SyncroTabs?

SyncroTabs is a video playback synchronizer across tabs.

It allows you to play and pause videos in different tabs with a single click.

How it works

  1. Click on the choose files button and select the videos you want to sync
  2. Ensure the number of files uploaded is correct
  3. Click on the open tabs button

FAQ

  • Why is only 1 tab created when I press open tabs => Make sure to enable this website to show pop up (disabled by default)
  • What is the maximum number of videos I can sync? => There is no limit.
  • Are the videos I upload being stored? => No, this website is not connected to any database.
